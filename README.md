# 🌱 Crop Recommendation System (Machine Learning + Streamlit)

## 🔍 Overview
This project predicts the most suitable crop based on soil nutrients and environmental conditions using a Random Forest machine learning model. It also provides an interactive web interface using Streamlit.

## ✨ Features
- Predicts best crop using ML model
- Interactive UI with Streamlit
- Displays confidence levels
- Feature importance visualization

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Plotly

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

3. Open browser:
   http://localhost:8501

## 📁 Project Structure
- app.py → Streamlit app
- train.py → model training
- model.pkl → trained model
